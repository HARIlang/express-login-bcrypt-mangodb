This is a sample project that contain express(node) as backend and mongo db as database 
This project is for my practice purpose for creating apis and handling logics
This project contain CRUD , signup and login with bcrypt 
The front-end part of the project is built with plain html and js it dose not contain any libraries or frameworks like react / angular

updated virson of this project is comes with react as a front end

packages used :
  
  backend : -
        express@5.2.1    cmd: npm i express        To handle the express project
        router@2.2.0     cmd: npm i router         To route the api in the backend modules
        dotenv           cmd: npm i dotenv         To use environmental variable in project
        cors@2.8.6       cmd: npm i cors           To used for api sharing data from client to server and server to client
        mongoose@9.5.0   cmd: npm i mongoose       To use database and to create model/schema of users 
        bcrypt@6.0.0     cmd :npm i bcrypt         Used for security purpose (ensuring the password of the user is safe);
express-rate-limit@8.4.1 cmd :npm i express-rate-limit    to limit the request 
   

front-end :

       axios           cdn: <script src="https://cdn.jsdelivr.net/npm/axios/dist/axios.min.js"></script>

       axios is used to transfer the data to backend
