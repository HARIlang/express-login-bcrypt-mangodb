# 🚀 Node.js + MongoDB Practice API Project

A full-stack practice project built to improve backend development skills using **Node.js**, **Express**, and **MongoDB**. This project focuses on creating secure APIs, handling backend logic, and integrating frontend applications.

Initially built with **plain HTML, CSS, and JavaScript**, the updated version now includes a **React frontend**.

---

## ✨ Features

- ✅ User Signup & Login Authentication
- ✅ Password Hashing with **bcrypt**
- ✅ Full **CRUD Operations**
- ✅ REST API Development
- ✅ MongoDB Integration
- ✅ Request Rate Limiting
- ✅ CORS Enabled
- ✅ Environment Variables Support
- ✅ Vanilla JavaScript Frontend
- ✅ Updated React Frontend

---

## 🛠️ Tech Stack

### 🔙 Backend

| Package | Install Command | Purpose |
|--------|----------------|---------|
| express@5.2.1 | `npm i express` | Backend server and API handling |
| router@2.2.0 | `npm i router` | Route handling |
| dotenv | `npm i dotenv` | Manage environment variables |
| cors@2.8.6 | `npm i cors` | Enable frontend-backend communication |
| mongoose@9.5.0 | `npm i mongoose` | MongoDB database integration |
| bcrypt@6.0.0 | `npm i bcrypt` | Password hashing |
| express-rate-limit@8.4.1 | `npm i express-rate-limit` | Prevent abuse / spam requests |

---

| Tool | Usage |
